视频字幕提取工具使用说明

1. 确保以下文件与程序在同一目录下：
   - ffmpeg-win32-x64.exe
   - wav2srt.exe
   - ggml-base.bin
   - app.ico
   - config.json (自动生成)

2. 运行程序，可通过以下方式选择视频文件：
   - 点击"选择视频"按钮浏览文件（默认打开上次选择的目录）
   - 直接将视频文件拖放到程序窗口

3. 选择需要的输出格式：
   - SRT字幕：标准字幕格式，包含时间信息
   - 纯文本字幕：仅包含字幕文本内容

4. 程序会自动保存你的选择，下次启动时会恢复

5. 点击"开始提取"按钮开始处理：
   - 程序首先获取视频信息和时长
   - 使用FFmpeg从视频中提取音频（进度条0-50%）
   - 使用wav2srt识别音频中的语音并生成字幕（进度条50-100%）
   - 处理过程会在日志窗口显示

6. 在处理过程中，可点击"停止转换"按钮终止操作

7. 处理完成后，字幕文件会保存在与视频相同的目录下，
   文件名为视频文件名加上相应扩展名。

注意事项：
- 处理时间取决于视频长度和计算机性能
- 请确保系统有足够的磁盘空间用于临时文件存储
- 如遇错误，请查看日志窗口的输出信息
